替换文件：
--------------------------------------------------------------------------------

把 EXE 重命名为“Update.exe”并复制到pubwin安装目录并替换如下文件：

   32位在 C:\Program Files\Hintsoft\Update.exe

   64位在 C:\Program Files (x86)\Hintsoft\Update.exe

   如遇到提示文件无法替换,则需要退出pubwin客户端再替换,或尝试修改pubwin安装目录的Update.exe文件名再替换

如未生效，请检查：
  清理服务器镜像分区缓存。
  多台服务器需要同步下镜像或者都替换。
